Progressive enhancement

Progressive enhancement is mostly when user without javascript can get good experience and with javascript even better one. It can also be applied to some modern CSS features.

It allows to reach more people, because javascript can be unavailable. Here is a good article about it: A handful of reasons JavaScript won't be available by Piccalilli.

To develop Progressive enhancement websites you want to start by implementing main functionality without javascript. Website should be working fine without it. After this gets done you can bring javascript to add additional helpers and so on. HTMX style (when you return partial html from server) can come very handy as you will be able to remove additional logic from your javascript.

Progressive enhancement not only will make your website accessible to more people, it also will make them simpler. You will less blow up your client with ton of javascript, because you will value your time to develop it. Reducing complexity can be called a side effect of Progressive enhancement. As well it will force your designer to not create some stupid, unrealistic and pixel perfect design.
